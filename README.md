# Web LuXe

[ Web LuXe: High-Dimensional Robust Inference ]

## Overview
Web LuXe is a high-dimensional probabilistic inference engine designed to analyze and monitor web responses with robust statistical models. Detects anomalies using multi-dimensional features, adaptive calibration, and probabilistic inference.

## Developer

## Features
- Multi-dimensional feature extraction (30+ features)
- Robust probabilistic inference (MinCovDet, Chi-square)
- Noise reduction via DOM analysis
- Adaptive calibration with outlier rejection
- Multi-system support: Windows, Linux, Termux, iSH

## Installation & Running

### Windows

git clone https://github.com/LUXEF7LK/WebLuXe.git



cd WebLuXe



pip install --upgrade pip



pip install -r requirements.txt



python main.py


### Linux

git clone https://github.com/LUXEF7LK/WebLuXe.git



cd WebLuXe



pip3 install --upgrade pip



pip3 install -r requirements.txt



python3 main.py


### Termux

pkg update && pkg upgrade -y



pkg install python git -y



git clone https://github.com/LUXEF7LK/WebLuXe.git



cd WebLuXe



pip install --upgrade pip



pip install -r requirements.txt



python main.py


### iSH (iOS Shell)

apk update && apk upgrade



apk add python3 py3-pip git -y



git clone https://github.com/LUXEF7LK/WebLuXe.git



cd WebLuXe



pip3 install --upgrade pip



pip3 install -r requirements.txt



python3 main.py
